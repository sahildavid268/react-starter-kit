React Starter Kit is a template for building web apps using react faremwork. 
It contains tools Webpack, Babel, SASS and Hot reloading. 

Prerequisite
System should have
- NODE (https://nodejs.org/en/) installed

- Installing Required Packages 
At root level of the project launch CMD and execute "npm i" or "npm install". 
This will install all the package and dependencies needed.

After installing the packages you are all set to go, 
run following commands at the root level to start the app.

Development Enviromnent 
npm run dev

Production Environment
npm run build
